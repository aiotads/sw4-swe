"Software engineering is programming integrated over time." 軟體工程，是把時間維度整合進來的程式設計。
完成進度：8 / 25 章
Time 時間

軟體工程是「整合了時間維度的程式設計」；程式碼有預期壽命，必須能因應變化。

Scale 規模

組織與程式碼長大時，每件重複做的事都要次線性擴展，否則撐不住。

Trade-offs 取捨

工程決策是在不完美資訊下權衡成本；要有理由，不能「因為我說了算」。

A close-reading map for L4–L6 software engineers
"Software engineering is programming integrated over time." In plain terms: software engineering = programming + the dimension of time.
Progress: 8 / 25 chapters
Time

Software engineering is "programming integrated over time"; code has an expected life span and must react to change.

Scale

As org and codebase grow, everything done repeatedly must scale sub-linearly, or it collapses.

Trade-offs

Engineering decisions weigh costs under imperfect information; they need reasons, never "because I said so."
